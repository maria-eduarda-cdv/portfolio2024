Trimeste 3:

Atividade Express
Descrição: Habilidades 1 e 2
Nessa atividade, criamos um novo projeto utilizando o Express e adicionamos o nodemon para facilitar o desenvolvimento. Configuramos um script chamado "serve" para rodar o projeto na porta 5000. Em seguida, criamos duas páginas de nossa escolha e adicionamos CSS para personalizar o visual das páginas, alterando a cor de fundo e a cor da fonte. Foi uma ótima oportunidade para praticarmos a criação de servidores web com Express e melhorar a apresentação das páginas com estilos.

Tarefa 02
Descrição: Habilidades 3 e 4
Nessa atividade, criamos um novo projeto que aceita pacotes externos e instalamos as bibliotecas inquirer e chalk. Utilizamos o inquirer para solicitar o nome e a idade do usuário e, em seguida, exibimos a resposta com um fundo amarelo e texto preto usando o chalk (com os métodos bgYellow e black). Além disso, adicionamos um tratamento de erro com catch para lidar com possíveis falhas durante a execução do inquirer. Essa atividade foi útil para praticarmos o uso de pacotes externos e melhorar o tratamento de erros em nossos projetos.
